# KeenKeeper

KeenKeeper is a simple React app to track and manage your friends and interactions.



## Features

* View all friends
* Add interactions (Call, Text, Video, Meeting)
* Timeline of activities
* Stats page with chart
* Toast notification
* Loading animation
* 404 Page for invalid routes


## Technologies

* React
* Tailwind CSS
* React Router
* Recharts
* React Toastify



##  Pages

* Home
* Friend Details
* Timeline
* Stats



##  How to Run

```bash
npm install
npm run dev
```
